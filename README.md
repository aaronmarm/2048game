## Instructions to Run Locally
- Clone Repository with the following command: <br />
`git clone github.com/aaronmarm/2048game`
- Run the following commands: <br />
```
cd 2048/src
javac Runner.java
java Runner
```
- Use the arrowkeys to play and enjoy!
